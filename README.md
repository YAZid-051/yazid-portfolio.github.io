# Portfolio 2025

Ce projet est un portfolio personnel réalisé en HTML et CSS.

## Aperçu

Le site présente :
- Mes compétences
- Mes expériences
- Mes formations
- Mes réalisations
- Mes certifications
- Ma veille
- Un formulaire de contact

## Accès en ligne

Le site est hébergé sur GitHub Pages :
- [Lien GitHub Pages](https://<ton-utilisateur>.github.io/<nom-du-repo>/)

Remplace `<ton-utilisateur>` et `<nom-du-repo>` par ton nom d'utilisateur GitHub et le nom du dépôt.

## Installation locale

1. Clone le dépôt :
   ```sh
   git clone https://github.com/<ton-utilisateur>/<nom-du-repo>.git
   ```
2. Ouvre le fichier `index.html` dans ton navigateur.

## Déploiement sur GitHub Pages

1. Pousse tous les fichiers sur la branche principale (`main` ou `master`).
2. Va dans les paramètres du dépôt > Pages.
3. Sélectionne la branche et le dossier racine (`/`).
4. Sauvegarde et accède à l’URL fournie.

## Structure et Description du Projet

### Fichiers Racine
- `index.html` : Page d'accueil du portfolio

### Dossiers
#### css/
- `sections.css` : Styles spécifiques pour chaque section du portfolio
- `style.css` : Styles généraux et mise en page principale

#### documents/
Ce dossier contient les documents complémentaires et ressources du portfolio

#### fichier/
- `Rapport de stage SNCF 2025 2.pdf` : Rapport détaillé du stage effectué à la SNCF

#### pages/
Pages principales du portfolio :
- `certifications.html` : Liste et détails de mes certifications professionnelles
- `competences.html` : Présentation de mes compétences techniques et professionnelles
- `contact.html` : Formulaire et informations de contact
- `experiences.html` : Parcours professionnel et expériences
- `formations.html` : Cursus académique et formations suivies
- `realisations.html` : Portfolio de projets et réalisations
- `veille.html` : Veille technologique et professionnelle
